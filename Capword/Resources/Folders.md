# Project folder structure

This folder contains the primary app source folders created by the assistant.

- Features/
  - Home/ - Home feature views and logic.
  - Words/ - Word listing and editor.
  - Profile/ - Profile and settings.
  - Onboarding/ - Onboarding flow and permission prompts.
- Core/ - App management, coordinators, environment.
- Services/
  - Network/ - Networking layer and API clients.
  - Permissions/ - Permission request wrappers.
  - Analytics/ - Analytics/telemetry integration.
- Shared/
  - Views/ - Reusable SwiftUI views (cards, pills).
  - Models/ - Shared domain models.
  - Utils/ - Utility extensions and helpers.
- Resources/ - Non-asset resources and documentation.

You can move files into these folders as features are implemented.
